# Octommerce
> Powerful E-commerce Engine built with OctoberCMS

## Why Octommerce?
- Easy to implement
- Flexible
- Modular

## Plugin Dependencies
- RainLab.User
- RainLab.Location
- Responsiv.Currency
- Responsiv.Pay

## Main Features
- Cart System
- Product Catalog
- Product Search
- Checkout
- Payment
- Taxation
- Multilanguage
